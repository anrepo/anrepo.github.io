Who is Frontend developer?

A specialist who works with visual part of a website, SPA or software.

The visual part also uses the abbreviation UI (User Interface), consists of developing basic markup, creating elements (buttons, forms, carousels, drop-down lists, etc.), and implementing the mechanics of user interaction with the interface (animations, interaction processing).

What does a Frontend developer do?

A front-end developer turns a design mockup created by a designer into a working user interface.

The developer can receive user data, process it, use it (mainly to transform the interface), and also transfer data to the backend.
