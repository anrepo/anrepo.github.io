Кто такой Frontend разработчик?

Специалист, занимающийся разработкой визуальной части web-сайта, SPA или програмного обеспечения.

Визуальная часть, так же использует аббревиатуру UI - User Interface, "Пользовательский интерфейс", складывается из разработки базовой разметки, создания элементов (кнопки, формы, карусели, выпадающие списки и пр.), реализации механики взаимодействия пользователя с интерфейсом (анимации, обработка взаимодействий).

Чем занимается Frontend разработчик?

Фронтенд-разработчик превращает дизайн-макет, созданный дизайнером, в рабочий пользовательский интерфейс.

Разработчик может получать пользовательские данные, обрабатывать их, использовать (в основном для преобразования интерфейса), а так же передавать данные на бэкенд.
